Tutorial

General

  1. Find your streamlabs socket API token in your streamlabs dashboard
    • Connect to streamlabs with Twitch account: Streamlabs website
    • Then go to Settings -> API settings -> API tokens -> "Your socket API token"
    • Copy the entire token (button "copy" on the right) Screenshot
  2. Paste it into "Streamlabs token" field in general configuration and then, configure your Jahy whatever you want to
  3. Click on "Generate configuration" button and copy-paste all the text in config.js
  4. In your OBS, add a new source and select "browser source" Screenshot
  5. Put thoses settings :
    • Local file : true
    • Local file : select the file "index.html" Screenshot
    • Width : 1245
    • Height : 760
    • Custom CSS : leave it blank
    • Shutdown source when not visible : true
    • Refresh browser when scene becomes active : true
  6. Resize and place whatever you want the Jahy in your scene Screenshot
  7. Enjoy !

Advanced

  • Base video: the video name of the base stand.
  • Video loop interval: Interval of time your base video loop.
  • Text: text displayed when the alert is triggered
  • Font size : Size of the text. If your text exceed the bubble, ajust the size
  • Bubble time : How fast the bubble will appear ("fade in" effect) *
  • Text time : How fast the text will appear after the bubble appear *
  • Active time : How long the text will stay displayed before fading out *
  • Name of the bubble : bubble image used for the alert *
  • Typography : Typography used for the alert *

* Bubble time + text time + active time + disappearance time (0.75 second) sould strictly always be smaller than the video animation time. Or else it will have a bug.

* Place your custom bubble image in "speechBubbles" folder. The size is 600x300 (W x H).

* Place your custom typography in "typo" folder.
