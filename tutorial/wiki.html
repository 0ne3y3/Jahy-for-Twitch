Tutorial

  1. Find your streamlabs socket API token in your streamlabs dashboard
    • Connect to streamlabs with your Twitch account: Streamlabs website
    • Then go to Settings -> API settings -> API tokens -> "Your socket API token"
    • Copy the entire token (with the button "copy" on the right)

    2. Streamlabs website: where to find your streamlabs token

  2. Paste it into "Streamlabs token" field in general configuration

    3. Streamlabs token field, in "Streamlabs" menu then "General" menu

  3. Configure your character whatever you want to, this wiki will help you to configure your character. See others sections
  4. Click on "Generate configuration" button

    5. The "Generate configuration" button

  5. Copy all the configuration generated by click the button "Copy". But also, save the width and height generated too, we will need it later for OBS

    6. The configuration to copy. Don't forget to note the width and height for OBS

  6. In your OBS, add a new source and select "browser source"

    7. OBS menu. Sorry it's in french

  7. Put thoses settings :
    • Local file : true
    • Local file : select the file "index.html"

      • the "index.html" file in window explorer

    • Width : *the width you noted earlier*
    • Height : *the height you noted earlier*
    • Custom CSS : leave it blank
    • Shutdown source when not visible : true
    • Refresh browser when scene becomes active : true

    The browser source menu

  8. Resize and place whatever you want the character in your OBS scene

    9. Resize and place your character. You can make it bigger or smaller, it still work

  9. Enjoy !

The general configuration menu

Use video or image

If you want to use a bundle of video or a bundle of image for the script. It seems logical that using videos use more CPU and RAM than images. Select the one you want to use by checking the appropriate button.

/!\ All your images or videos must be at the same width and height.

Place speech bubble

Place the speech bubble on the right or on the left of the character.

When using a reduced gap, if you place the speech bubble on the left, your character will be on top. While placing the speech bubble on the right, it's the speech bubble which will be on the top.

You can see the difference between placing on the right or on the left

Well, it's the color of the text in the speech bubble. Have to be an hexadecimal.

Shadow on text

If you want to add a shadow for your text.

Shadow exemple

Base video or Base image

The video or image displayed when there is no alert.

/!\ For videos, it need to loop at regular interval, see "video loop interval" to understand why. It's not mandatory, but it will give the vibe your character is really reacting to the alert.

Video loop interval

The interval when the script check if an alert arrived.

For image : it can be 2/3 seconds, we don't really care since it's not a loop. Avoid small number like 1 second, it's not cpu usage friendly.

For video : I recommend 5 seconds minimal to 10 seconds maximal. 5 seconds minimal will let you some time to do an animation between those check. 10 seconds maximal, because if more, your character will react really late if an alert arrive 6/7 seconds before the check.

/!\ For Jahy-sama, keep this value to 5.

Margin text top

Change this value to place your text inside your speech bubble.

For my bubble speech, my margin is 40 pixels

Center text

The size of your text container. Your text will always be centered to the speech bubble. Long username will overflow from the text container. So keep in mind to make your speech bubble wide.

For my bubble speech, I choose 70 pixels

Gap

The gap between your speech bubble and your character. Negative numbers will inscrease the gap. Change this value if you think your character is too far of the bubble speech.

The gap between your character and the speech bubble

Advanced configuration

The advanced configuration menu

Activate text

If you want to activate, or not, the bubble speech for this alert.

[Follow/Sub/etc...] text

The text displayed when the alert is triggered. ${variable} will be replaced by the username, amount of bits, gifter, etc...

Font size

The font size of the text displayed when the alert is triggered.

Bubble time - Text time - Active time

Bubble time : how long the bubble will take to appear, with a "fade in" effect. Text time : how long the text will take to appear, with a "transform" effect. Active time : how long the text will be displayed before "fade out" with the bubble. Disappear time : the bubble and text will disappear in 0.75 seconds, with a "fade out" effect.

If you don't want the animation, set bubble time and text time to 0.

The bubble animation

Name of the bubble

File name of the speech bubble to display when the alert is triggered. Speech bubbles have to be in the "speechBubbles" folder.

/!\All your speech bubble need to be the same width.

Typography

File name of the typography to use when the alert is triggered. Typography have to be in "typo" folder.

Video name or Image name

File name of the video/image to display when the alert is triggered. videos/images have to be in "video" folder.

Add variation

If you want to add some variation to this alert, the script will choose randomly between your variations and display its animation.

Twitch tutorial

  1. First you need to create a Twitch account for your bot. You can use your own account but I don't know if Twitch considerate it "illegal"
  2. Then you need to get the Oauth password, because bots account doesn't use normal password
    • Connect to twitch app with your bot account : TMI website
    • Authorize the application to connect with your account
    • Copy all the password

    3. The TMI website

  3. In the configuration page, go to "Twitch chat" section => "General". Activate the twitch chat option

    4. The twitch general menu

  4. Fill the fields:
    1. Your twitch bot account name
    2. The Oauth password generated earlier. Don't forget to copy "oauth:" with it.
    3. Your twitch channel (where you stream)
  5. It's already done, see others section to configure regex and mini games.

Twitch chat

The twitch chat menu

Option

Choose if the keyword have to be contained in the sentence, or if it have to be the first word of the sentence. For exemple we take 3 sentences :

1) "It's super ! PogChamp"

2) "PogChamp"

3) "PogChamp super cool !"

If the keyword is "PogChamp" and the option is "contain", then all those 3 sentences will trigger the animation. If it's "start with", only the second and third sentence will trigger the animation.

You'll probably only use "start with" when it's for a command like "!twitter".

Word

The word that will trigger the animation. Small tip, it can be a sentence. Words are case insensitive so "pogchamp" = "PogChamp" = "POGCHAMP".

/!\ Any Unicode characters are available except: "/\ [] () {} ? + * | ."... To use those, you need to place \ before. For exemple "money \$\$\$" for "money $$$".

Timeout

When the alert is triggered, wait X min before the alert is available again.

Commands

If you want to add an action to this command (like writting a message in chat, timeout the user, etc...).
